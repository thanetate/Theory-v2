# Theory Climbing
Check it out here: 🔗 https://www.theoryclimbing.com/
### Overview
As an avid rock climber with over 7 years of experience, I’ve always dreamed of creating my own clothing brand. This project has allowed me to finally bring that vision to life.

Theory Climbing is the result of my passion for both climbing and web development. The website was built using React and ASP.NET Core, also including Supabase for secure authentication and the Stripe API for a smooth checkout experience.

This is version 2.0 of the website. Originally built with Next.js, I wanted to challenge myself and improve the project by starting from scratch, rebuilding it with a more refined tech stack and a fresh approach.
