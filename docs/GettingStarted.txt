Getting Started ->

    1. Run the Client
        npm run dev -> port 3000 
        npm run build -> build the wwwroot file for the server

    2. Run the Server
        dotnet run -> port 5225
    
    3. Set up Environment Variables
        Client (.env) ->
            VITE_SUPABASE_URL
            VITE_SUPABASE_KEY
            
        Server (appsettings.json) ->
            AllowedHosts
            SupabaseUrl
            SupabaseKey
            ValidIssuer
            ValidAudience
            JwtSecret
            StripeSecretKey
        Server (secrets.json) ->
            Authentication:JwtSecret
