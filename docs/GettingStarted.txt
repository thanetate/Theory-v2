## This is the general outline of the app

Tech Stack -
    1. Frontend: React
            - Vite (TypeScript)
            - Jotai (State Management Library)

    2. Backend: C#, .NET
    3. Database: PostgreSQL
            - SupaBase


Needs -

    4. Security and Scalability

Phone -

        h2 -> 14px normal 400
Desktop -
        h1 -> 24px normal 400 (option - opacity 1)
        h2 -> 14px normal 400

Client -
        1. npm create vite@latest .
        2. change port 5174 -> 3000
        3. npm install

Server - 
        1. dotnet new webapi
        2. dotnet run
        3. test -> curl curl http://localhost:5255/weatherforecast/
        4. add CORS
        5. make sure to add proxy to vite config
        6. Connect to api in frontend 
        7. use Axios to get endpoint from backend
